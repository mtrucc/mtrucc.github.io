“华为畅玩4”安卓手机使用感受

发表于 2017-12-09 更新于 2018-12-03 分类于 Android
本文字数： 1.1k 阅读时长 ≈ 1 分钟

我有3个电信卡，所以我最近要弄一台备用的手机。我在转转那找到一台华为畅玩4的安卓手机，同城面交，花了259元。我说一下我用这台手机的感受吧。

声明

这是使用3年iPhone6之后和259元买的“华为”安卓机做对比后的感受，不能作为参考！
这篇文章写于2017年12月9日，这时候的iPhone6和华为畅玩4都已经算是老手机了！
如果对本文有哪方面不同意或者有看法和意见，请在下方留言。无法留言的请自备梯再上本站。

初试

拿到手机，我也懒得刷机，直接恢复了出厂设置。因为我看那卖家应该是个会玩的，刷的基带应该挺稳，系统也应该是最新的。
恢复好系统后，出厂设置好后，开始使用。

连接WIFI

我们宿舍有支持5G WIFI的路由器，这台手机无线模块是不支持的，所以搜不到信号，后来连的2.4G WIFI的信号。

安装APP

安装APP用的华为应用商店，界面看上去有点。。乱，复杂，有点广告扎堆的感觉。
苹果也有这个现象，但并不是很严重。

想了一下，懒得安装APP了，我这个用来接短信接电话就行了。

界面UI

看到主屏幕，恢复系统后自带的一堆用不到的APP，心塞得很，也懒得排版布置，以及~卸载。

长时间使用

长时间使用的话，头特别疼，可能是由俭入奢易由奢入俭难把。头太疼了不能用这个安卓手机了。

总体感受

APP商店：贵圈太乱
手机管理：细节做的太多了，各种花式弹窗。对于爱折腾的同学来说，各种DIY也是不错的
长时间使用的话，头会很痛很痛。

重点

这是使用3年iPhone6之后和259元买的“华为”安卓机做对比后的感受，不能作为参考！

建议

  1. 不喜欢折腾的同学还是用苹果把，或者稍微正常点的安卓机。
  2. 给你的父母优先使用iPhone5 or iPhone5s，太新的安卓机，我觉得他们用着会很难受有风险。
  3. 如果你想使用安卓系统的时候有更好的感受，请买高端安卓机。
  4. 不要买1000 2000的安卓机了，一分钱一分货。可能你新买到的1000元的手机不如已经过时的iPhone5s。我同学就是个例子，手机很新，但是却搜不到WIFI的5G信号，他们在WIFI模块那省了一些钱~~

后记

  • 用安卓低端机，严重拉低了我对安卓系统和安卓手机的评价和印象，我的感觉非常非常的不好。
  • 我并不是针对华为，针对安卓系统和安卓手机，我只是客观的说出我的感受，花粉和各位使用安卓的朋友不要误会。
  • 期望后面我能用到好一点的安卓机，我要换手机了（目前使用的是iPhone6，它已经陪伴我3年了），期望买到稍微好一点的手机，我正在安卓和苹果之间犹豫不决。
  • 这篇文章并没有结束，只是我的感受太多（外加头疼！！！！）并且难以形容，我就先写到这了。
