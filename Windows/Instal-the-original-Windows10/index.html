不用WINPE安装纯净的Windows 10

发表于 2018-07-04 更新于 2018-12-03 分类于 Windows
本文字数： 980 阅读时长 ≈ 1 分钟

就在昨天，我从老板那儿Get了个技能，非常有用。
喜欢折腾的技术宅，多多少少会自己重装“干净”的系统，但是网上下载的大白菜啊，老毛桃啊，总是在你装完系统后加装一些奇奇怪怪的软件，非常讨厌。
然后网上的一键装机也都阉割了很多组件，导致不少软件都不能够正常的使用，非常烦人。

准备工作

一个可以正常使用的Windows操作系统和一个8G以上大小的U盘，这个就不必多说了把。

下载官方原版镜像

在这儿可以下到官方原版镜像：MSND，我告诉你

装载原版ISO映象

右键点击下载好的ISO映象文件 - 装载
如果当前 Windows 10 已安装的第三方软件已关联了.iso文件，则无法通过双击直接挂载，右击之后也不会出现「装载」选项。此时可以在右击弹出的快捷菜单中通打开方式—Windows 资源管理器来手动装载 ISO。
ISO 装载好之后 Windows 10 会自动创建一个虚拟光驱并分配一个盘符，我们可以通过左侧窗格或者我的电脑直接导航并查看到其中的内容。

格式化U盘，拷贝安装文件到U盘

然后我们需要先格式化U盘
先按下键盘组合键win + r打开运行窗口，输入cmd打开Windows命令处理程序。

  • 使用diskpart命令分区。
  • list disk列出当前可用分区
  • select disk xx选择磁盘，xxx代表磁盘号，选择U盘即可
  • clean清除磁盘
  • convert gpt转换为gpt格式
    创建EFI分区及系统安装分区：
    create partition efi size=100 创建EFI分区，大小100M
    create partition msr size=128 创建MSR分区，默认是 128M

让你的电脑从U盘启动

这个不好详细说，具体请百度：“电脑型号 + U盘启动”

u盘启动后进入安装界面不要急着安装

首先点击开始安装后按shift + f10组合键

  • 使用diskpart命令分区。
  • list disk列出当前可用分区
  • select disk xx选择磁盘，xxx代表磁盘号，选择U盘即可
  • clean清除磁盘
  • convert gpt转换为gpt格式
    然后创建EFI分区及系统安装分区：
    create partition efi size=100创建EFI分区，大小100M
    create partition msr size=128创建MSR分区，默认是 128M

开始安装

按照提示正常安装即可
