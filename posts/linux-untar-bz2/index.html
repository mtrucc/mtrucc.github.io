Linux 下解压.tar.bz2 压缩包

Published:Dec 11, 2017 |  at 12:59 AM | Suggest Changes

今天我下载了一个文件，是 .tar.bz2 格式的 但是我用在网上找的解压命令无论怎么样都无法解压，耐心查看了一下错误输出信息，原来缺少 bzip2 组件导致的。

安装 bzip2 组件

Centos 系统运行下面的命令安装 bzip2：

yum install bzip2 -y
apt-get -y install

Debian/Ubuntu 则用下面的命令安装 bzip2：

yum install bzip2 -y
apt-get -y install

解压命令

老版本的 linux 要两步，一步是解压缩，一步是解包

bzip2 -d file.tar.bz2 ##将文件解压成 file.tar
tar -xvf file.tar ##解包

解完之后会出现多一个文件夹 file 或者直接执行下面的命令解压

tar -jxvf file.tar
